Important HTML Tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1-h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table includes a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make strong text <strong>at least a billion people</strong> It bolds the text
a anchor make hyperlink <a href="https://learnenough.com/">Learn Enough</a> It changes a word or phrase into a link
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl"> it allows an image to be placed on a website
code code syntax show source code This is source code This is source code
div division Divide content
span span Make content inline with the surrounding content
blockquote blockquote Include quotation
header header Isolate the head text
br break Breaks the line

Document Tgs

Tag Name Purpose
html html Include html code
head head Include a head text
body body Include body of data
title title Include the page title
meta meta Include defination of characters
